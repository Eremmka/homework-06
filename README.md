# HomeWork 06-0
Создание приложения по тематике мультфильма рика и морти.
## Задача
Написать приложение в тематике рика и морти.
Сделать маршрутизацию.
Сделать три страницы с информацией, в которых должны быть объекты в с ссылками уже на другие страницы.
## Описание
В главном компоненте App написаны все Route, там же происходит переход к первой страничке Home.
Все обёрнуто в HeaderLayout, для того чтоы отражались навигационные кнопки. Обернуто все кроме Route для страницы NotFound.
При переходе в категорию отображается список элементов, которые относятся к этой категории(каждая категория имеет свой маршрут), при нажатии на элемент происходит переход 
на страничку с подробной информацией элемента(маршрут также меняется при переходе от элемента к элементу).
Routes прописан в главном компоненте App. Все Route прописаны в одном месте. Все приложение имеет обычный не особо замысловатый стиль, но по тематике мультфильма.
## Иструменты и библиотеки

1. JavaScript
2. React
3. React-Router v6
